# ブロック崩しゲーム

アルカノイド風のブロック崩しゲームです。

## プレイ

[https://yoshiken2k-sys.github.io/breakout/breakout.html](https://yoshiken2k-sys.github.io/breakout/breakout.html)

## 操作方法

| 操作 | 内容 |
|---|---|
| マウス移動 | パドルを左右に動かす |
| 左クリック | 弾を発射 / キャッチ中のボールを解放 |
| ← → キー | パドルを左右に動かす |
| スペースキー | キャッチ中のボールを解放 / リスタート |

## アイテム一覧

ブロックを壊すと一定確率でアイテムが落下します。パドルで取得すると効果が発動します。  
新しいアイテムを取ると前の効果は解除されます。

| アイテム | 色 | 効果 |
|---|---|---|
| `3×` | 紫 | ボールが3つに分裂 |
| `C` | 水色 | パドルでボールをキャッチ（クリックで解放） |
| `P` | 橙赤 | ボールがブロックを貫通（10秒間） |
| `W+` | 緑 | パドルが拡大（10秒間） |
| `W-` | ピンク | パドルが縮小（10秒間） |
